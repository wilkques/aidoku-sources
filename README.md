# Aidoku Wilkques Sources

This repository hosts unofficial sources maintained by community members that are installable in [Aidoku](https://github.com/Aidoku/Aidoku).

## Usage

On a device with Aidoku (0.7+) installed

Navigate to the settings tab, and under the source lists page add `https://raw.githubusercontent.com/wilkques/aidoku-sources/refs/heads/sources/index.min.json`.
